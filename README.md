# Next-Portfolio-2.0

A modern, interactive portfolio built with Next.js, React, and Tailwind CSS. Features an interactive terminal, games, and a visually stunning design.

## Features
- Terminal UI with interactive commands
- Built-in CLI Games (Tic-Tac-Toe, Number Guessing)
- Dynamic animations using GSAP
- Fully responsive design

## Setup
1. Clone the repository
2. Run `npm install` or `pnpm install`
3. Start the dev server `npm run dev`

## Technologies Used
- Next.js
- React
- TypeScript
- Tailwind CSS
- GSAP
